# Mewtwo No Rarity Hunting Guide

*Last Updated: February 1, 2026*

The Mewtwo No Rarity (マークなし/初版) is from the 1996 Japanese Base Set first print run, before rarity symbols were added. Look for cards WITHOUT a star symbol in the bottom right corner.
